# English Labradors Hub

## Overview

**English Labradors Hub** is a professional website dedicated to connecting families with premium English Labrador Retrievers from the heart of Florida kennels. The platform showcases available puppies, provides detailed information about the breed, and offers a seamless experience for prospective owners to request availability and learn about champion bloodlines.

## Features

- **Responsive Design**: Fully optimized layout for seamless browsing on desktops, tablets, and mobile devices.
- **Dynamic Animations**: Engaging user experience with GSAP, AOS, and WOW.js animations.
- **Interactive Galleries**: Fancybox integration for viewing puppy photos and kennel images.
- **Testimonial Sliders**: Slick carousel showcasing satisfied client reviews.
- **Availability Request**: Dedicated form for prospective owners to inquire about puppies.
- **Modern Aesthetics**: Clean, professional design with a focus on visual appeal and usability.

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Framework**: Bootstrap 5, jQuery 3.6.0
- **Animations**: GSAP (GreenSock), ScrollSmoother, ScrollTrigger, AOS (Animate On Scroll), WOW.js, Animate.css
- **Libraries**: Slick Carousel, FancyBox, SlickNav, Font Awesome 6.7.2
- **Fonts**: Custom fonts (AmpleSoftPro)

## Key Sections

**Home**: Hero banner with kennel introduction, about section, available puppies preview, mission statement, gallery, and contact form.

**About Us**: Detailed information about the kennel, breeding philosophy, and commitment to quality.

**Our Labradors**: Comprehensive breed information, characteristics, and what makes English Labradors special.

**Available Puppies**: Current litter listings with photos, pricing, and availability status.

**Testimonials**: Reviews and feedback from satisfied families who have adopted puppies.

**Contact**: Multiple contact methods including phone, email, and location information.

## Installation

1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. No backend or database installation is required for the static interface.
