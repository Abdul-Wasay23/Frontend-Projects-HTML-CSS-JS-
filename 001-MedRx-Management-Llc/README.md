# Integrated Mind & MedRx Management LLC

## Overview
This repository contains the source code for the MedRx Management LLC website. Integrated Mind & MedRx Management LLC specializes in providing comprehensive mental health services, including anxiety and depression treatment, substance use disorder management, and crisis support.

The website serves as a platform to inform patients about services, provide mental health resources, and facilitate appointment bookings.

## Features
-   **Home Page**: Overview of services, mission/vision, and client testimonials.
-   **About Us**: Information about the team and the organization's mission.
-   **Services**: Detailed descriptions of treatments for Anxiety, Depression, ADHD, and Crisis Support.
-   **Appointment Booking**: Easy access to book appointments.
-   **Blog**: Educational resources and articles on mental health.
-   **Responsive Design**: Optimized for desktop, tablet, and mobile devices.

## Technologies Used
-   **HTML5**
-   **CSS3**: Custom styling, Bootstrap, SlickNav, Fancybox, Animate.css, AOS (Animate On Scroll).
-   **JavaScript**: jQuery, GSAP (GreenSock Animation Platform) for animations, Slick Slider.

## Project Structure
-   `index.html`: Main landing page.
-   `aboutus.html`: Organization information.
-   `services.html` & `service-detail.html`: Service listings and details.
-   `blogs.html` & `blog-detail.html`: Blog section.
-   `appointment.html`: Appointment booking page.
-   `css/`: Stylesheets (Bootstrap, custom CSS, etc.).
-   `js/`: JavaScript files (jQuery, custom scripts, plugins).
-   `images/`: graphical assets.
-   `slick/`: Slick carousel assets.

## Usage
To view the website locally:
1.  Clone this repository or download the source code.
2.  Open the project folder.
3.  Double-click `index.html` to open it in your web browser.
