# WinVerse

**WinVerse** is a premium online competition platform featuring real-time ticket tracking, instant wins, and a gamified checkout experience.

## Features

- **Responsive & Modern**: Optimized for all devices with high-end animations (GSAP, AOS).
- **Gamified Experience**: Includes "Spin the Wheel" and scratch cards post-purchase.
- **Real-Time Tracking**: Live updates on sold tickets and draw status.
- **Review System**: Trustpilot integration and testimonial sliders.
- **Secure Checkout**: Complete cart system with discount codes and secure payment UI.

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (jQuery 3.6.0)
- **Framework**: Bootstrap 5
- **Animations**: GSAP, ScrollSmoother, AOS, WOW.js
- **Plugins**: Slick Carousel, FancyBox, NoUiSlider

## Installation

1. Clone the repository.
2. Open `index.html` in your browser.
3. No backend setup required.
