# Leon Nutrition - Premium Nutritional Products Platform

## Overview

**Leon Nutrition** is a modern e-commerce platform dedicated to providing high-quality nutritional supplements and wellness products. It offers a wide range of products including protein powders, dietary supplements, wellness accessories, and health-focused gear, designed to support your fitness journey and overall wellness goals.

## Features

- **Responsive Design**: Fully optimized layout for seamless browsing on desktops, tablets, and mobile devices.
- **Dynamic Animations**: Engaging user experience with GSAP, AOS, and WOW.js animations.
- **Product Filtering**: Easy-to-use filters for categories, price, and product attributes.
- **Interactive UI**: Slick sliders for products and testimonials, fancybox for image viewing.
- **Shopping Features**: Complete cart and checkout flow, wishlist functionality, and account management.
- **Modern Aesthetics**: Clean, professional design with a focus on visual appeal and usability.
- **Customer Testimonials**: Display of customer reviews and success stories.
- **FAQ Section**: Comprehensive frequently asked questions about products and services.

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Framework**: Bootstrap 5, jQuery 3.6.0
- **Animations**: GSAP (GreenSock), ScrollSmoother, AOS (Animate On Scroll), WOW.js, Animate.css
- **Libraries**: Slick Carousel, FancyBox, Font Awesome 6.7.2
- **Fonts**: Custom fonts (Barlow, Montserrat, Poppins, Roboto)

## Installation

1. Download or clone the repository from GitHub.
2. Open `index.html` in any modern web browser.
3. No backend or database installation is required for the static interface.
4. For best experience, use a modern browser (Chrome, Firefox, Safari, Edge).


### Animations
- GSAP animations: `js/gsap-animation.js`
- AOS settings: `js/custom.js`
