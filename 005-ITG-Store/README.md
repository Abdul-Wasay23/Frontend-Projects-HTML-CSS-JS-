# ITG Store - E-Commerce Platform

## Overview

**ITG Store** is a modern e-commerce platform dedicated to providing high-quality products and accessories. It offers a wide range of items designed to optimize your lifestyle and enhance your experience.

## Features

- **Responsive Design**: Fully optimized layout for seamless browsing on desktops, tablets, and mobile devices.
- **Dynamic Animations**: Engaging user experience with GSAP, ScrollSmoother, WOW.js, and Animate.css animations.
- **Product Filtering**: Easy-to-use filters for categories, price, and attributes like Hot Deals, What's New, Daily Deals, Best Sellers, Clearance.
- **Interactive UI**: Slick sliders for products and testimonials, FancyBox for image viewing.
- **Shopping Features**: Complete cart and checkout flow, wishlist functionality, and account management.
- **Modern Aesthetics**: Clean, professional design with a focus on visual appeal and usability.

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Framework**: Bootstrap 5, jQuery 3.6.0
- **Animations**: GSAP (GreenSock), ScrollSmoother, WOW.js, Animate.css
- **Libraries**: Slick Carousel, FancyBox, Font Awesome 6
- **Fonts**: Custom fonts (Ample Soft Pro)

## Product Categories

**Apparel**: Various clothing items

## Installation

1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. No backend or database installation is required for the static interface.
