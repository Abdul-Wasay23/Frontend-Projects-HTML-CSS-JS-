# Marriage Growth Summit - Expert-Led Marriage Masterclasses

## Overview

Marriage Growth Summit is a premium, faith-based educational platform dedicated to empowering marriages through expert-led masterclasses and supportive community resources. Built with modern web technologies, it provides couples with transformative sessions focused on strengthening relationships and building lasting partnerships.

## Features

- **Expert-Led Masterclasses**: Professionally curated marriage enrichment sessions
- **Responsive Design**: Fully optimized for all devices
- **Interactive Animations**: Smooth GSAP, AOS, and WOW.js scroll animations
- **Client Testimonials**: Slick Carousel powered testimonial slider
- **Counter Statistics**: Animated stats showcasing experience and success

## Technology Stack

| Category | Technologies |
|----------|--------------|
| **Core** | HTML5, CSS3, JavaScript |
| **Framework** | Bootstrap 5, jQuery 3.6.0 |
| **Animations** | GSAP, AOS, WOW.js, Animate.css |
| **Components** | Slick Carousel, FancyBox, SlickNav |
| **Icons** | Font Awesome 6 |
