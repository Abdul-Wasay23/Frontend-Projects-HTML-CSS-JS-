# Mobile Master - Mobile Components E-Commerce Platform

## Overview

**Mobile Master** is a modern e-commerce platform for mobile phone components and repair parts. It specializes in mobile panels, LEDs, batteries, and repair components for iPhone, Samsung, Vivo, Oppo, and Huawei devices.

## Features

- **Advanced Filtering**: Filter by category, phone brand, quality, and price range
- **Responsive Design**: Mobile-optimized for all devices
- **Interactive Product Cards**: Color selection, cart, wishlist, quick view
- **Multi-Language Support**: Google Translate integration
- **Promotions**: Daily deals (70% off) and seasonal sales
