# Spirit-X - School Spirit Wear Platform

## Overview

**Spirit-X** is a modern e-commerce platform dedicated to providing high-quality school spirit wear and accessories. It offers a wide range of apparel including hoodies, personalized jerseys, caps, and team gear, designed to boost school pride and community spirit.

## Features

- **Responsive Design**: Fully optimized layout for seamless browsing on desktops, tablets, and mobile devices.
- **Dynamic Animations**: Engaging user experience with GSAP, AOS, and WOW.js animations.
- **Product Filtering**: Easy-to-use filters for categories, price, and attributes.
- **Interactive UI**: Slick sliders for products and testimonials, fancybox for image viewing.
- **Shopping Features**: Complete cart and checkout flow, wishlist functionality, and account management.
- **Modern Aesthetics**: Clean, professional design with a focus on visual appeal and usability.

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Framework**: Bootstrap 5, jQuery 3.6.0
- **Animations**: GSAP (GreenSock), ScrollSmoother, AOS (Animate On Scroll), WOW.js, Animate.css
- **Libraries**: Slick Carousel, FancyBox, Font Awesome 6
- **Fonts**: Custom fonts (Cutmark, Montserrat)

## Product Categories

**Apparel**: Hoodies, T-Shirts, Jerseys, Jackets

**Accessories**: Caps, Bags, Scarves

**Collections**: Men, Women, Kids, Unisex

**Features**: Customizable prints, various sizes (S-XL), multiple color options.

## Installation

1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. No backend or database installation is required for the static interface.
